Django backend online

Hyperthon backend and frontend are separated cleanly.

This Django app powers vehicle inventory, authentication, and user registration APIs. The React frontend now lives in the separate frontend folder and communicates with this backend through JWT and vehicle endpoints.

Open Django Admin View Vehicle API JWT Login Endpoint

Marketplace API

GET /vehicle/ for the inventory list, and filter by brand, horsepower, speed, price, or year.

Authentication

POST /api/token/ to sign in from the frontend and attach the returned access token to protected calls.

Management

Use the frontend login and marketplace form to create vehicles after signing in.
